1.Bootstrap 5 มีอะไรใหม่ๆบ้างที่แตกต่างจาก Version เก่า อย่างน้อย 5 ตัวอย่าง

ไม่มีการใช้งาน JQuery ใช้ Pure JavaScript ที่เบาและเร็วกว่า
Container เป็นแบบ Responsive
Sticky Top เป็นแบบ Responsive
เปลี่ยนมาใช้ Icon แบบ SVG แทน Font Icon
ปรับส่วนของ Gird เป็นแบบ FlexBox และ Grid Layout


2.จุดเด่นของ Bootstrap คืออะไร จงอธิบาย

มีระบบ Grid มาช่วยและมีการคำนวนค่าหน้าจอพร้อมกับปรับขนาดของ Web ให้แสดงผลกับทุกๆ หน้าจอโดยอัตโนมัติ
สามารถปรับแต่งให้แต่ละหน้าจอแสดงผลต่างๆกันได้ ตามขนาดของหน้าจอ
มีเครื่องมือที่พร้อมสนับสนุนการทำงาน และมีรูปแบบที่สามารถทำความเข้าใจได้ง่าย


3.จงอธิบายการติดตั้งและใช้งานแบบ CDN และเเบบ Internal (Compiled CSS and JS )แตกต่างกัน อย่างไร

แบบ CDN ต้องใช้อินเทอร์เน็ตเพื่อใช้งานแต่ต้องโหลดอะไรเลย
ส่วนแบบ Internal ต้องโหลดไฟล์เพื่อใช้งานแต่ไม่จำเป็นต้องมีอินเทอร์เน็ต
